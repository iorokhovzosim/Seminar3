## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элеменнт 2
* Элемент 3

Чтобы добавить нумированные списки, необходимо просто пронумеровать(1). Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
